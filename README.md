# Fresh Bite Cafe

A modern and responsive restaurant website built using React, TypeScript, and Vite. Fresh Bite Cafe offers a clean and attractive user interface with sections like Home, About, Menu, Gallery, and Contact to provide a smooth user experience.

## Live Demo

https://fresh-bite-cafe.vercel.app/

## GitHub Repository

https://github.com/Deekshitha-28/fresh-bite-cafe

## Features

- Responsive modern UI
- Smooth navigation
- Attractive landing page
- Menu showcase section
- Gallery section
- Contact section
- Fast loading performance
- Reusable React components

## Tech Stack

- React
- TypeScript
- Vite
- CSS

## Installation

Clone the repository:

```bash
git clone https://github.com/Deekshitha-28/fresh-bite-cafe.git
```

Navigate into the project folder:

```bash
cd fresh-bite-cafe
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

## Build for Production

```bash
npm run build
```

## Preview Production Build

```bash
npm run preview
```
